## ⚡ Architecture & Tech Stack

* **Framework:** Next.js (App Router)
* **Language:** TypeScript (Strictly typed)
* **Styling:** Tailwind CSS
* **Content Management:** Sanity.io (Headless CMS)
* **Deployment & Analytics:** Vercel

## ✨ Key Features

* **Headless CMS Integration:** All content (projects, timeline, text) is managed dynamically via Sanity Studio, triggering automated Vercel rebuilds via Webhooks upon publishing.
* **Flawless Performance:** Achieved straight 100s on Google Lighthouse audits for Performance, Accessibility, Best Practices, and SEO.
* **Strict Type Safety:** Fully typed across the entire stack, eliminating runtime errors and ensuring data integrity between the Sanity backend and Next.js frontend.
* **Optimized Image Pipeline:** Utilizes Next.js `<Image>` component coupled with Sanity's image CDN for automatic WebP conversion, resizing, and lazy loading.
